📘 Student Grade Calculator

A simple and efficient Student Grade Calculator that helps compute total marks, percentage, and final grade based on user input. This project is ideal for beginners learning programming fundamentals such as variables, conditionals, and user input handling.

🚀 Features

Input marks for multiple subjects

Calculate:

Total marks

Average percentage

Grade (A, B, C, etc.)

User-friendly interface (CLI or GUI depending on implementation)

Error handling for invalid inputs

Customizable grading system

🧮 Grading Criteria (Example)
Percentage	Grade
90–100	A
75–89	B
60–74	C
50–59	D
Below 50	F
🛠️ Technologies Used

Language: (e.g., Python / Java / C++ / JavaScript)

Optional: HTML, CSS (for web version)

📂 How It Works

User enters the number of subjects

Inputs marks for each subject

Program calculates total and average

Displays the final grade based on percentage

▶️ Example Output
Enter number of subjects: 5
Enter marks: 85, 90, 78, 88, 92

Total Marks: 433
Percentage: 86.6%
Grade: B
